== Installation ==

Instructions:

Before you begin, head on over to our website in order to create an account if you haven't already done so https://socialproofus.com/register

Your PIXEL key can be found by:

1. Going to your SocialProofus dashboard
2. Select "Latest campaigns" 
3. Click the 3 dots to the right of the campaign. 
4. Click "Install PIXEL".
5. Copy the 32 character PIXEL code found at the end of the URL e.g. https://socialproofus.com/pixel/hta3gsej50fyonyl18twvm1xufvnwzkd

1. Install the plugin through the WordPress plugins screen OR unzip the plugin archive to '/wp-content/plugins'
2. In the WordPress 'Plugins' page, activate the plugin.
3. Select "SocialProofus" menu item from your admin sidebar.
4. Copy over your PIXEL key from SocialProofus dashboard and hit "Save".
5. That is it for the installation and all you need to do is setup your widgets.

Notice:

* SocialProofus will inject JavaScript code to your website to display notifications and collect visitor data.
* Management is available via our website (click the logo in the settings page).
